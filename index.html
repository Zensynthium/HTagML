Welcome to the HTagML (HTML Tag Challenge)!

A Brief Introduction

This is actually going to be quite the extreme gauntlet actually doable project. We're going to incorporate every HTML tag 50 tags into a single page in a orginal and unique fashion to demonstrate complete relative mastery of HTML5, while keeping proper website and code structure principles in mind! Feel free to glance into the console using the appropriate shortcut (i.e. ctrl + shift + j in Google Chrome on Chrome OS/Windows), or download the document to ensure that you get an inside look at how the page is constructed as well!

Rules

  1. Tags <h1> - <h6> will count as individual tags.
  2. Each entry must pass through the appropriate validators (html, css, javascript, etc.).
  3. Have fun! :^)

Without further ado, here we go!

HTagML has rather humble origins, as it was thought up as I was putting off homework and decided that maybe there was more fun to be had experimenting with Front End Web Development than droning away at my homework (not that I'm condoning procrastination or anything). I discovered that there were over 100 tags (many of them being very specific) and thought it would be interesting to see how others would try and meet the requirement of trying to cram approximately half of them into a single document! I immediately strayed away from the idea of using every single tag as I quickly realized at the pace I was going it would take roughly ∞∞ years to complete...a bit longer than my goldfish-like attention span has to spare.
